Demostração

D7data sistema de gestão para negócios (ERP) de informática e assistência técnica. Ele facilita o controle de estoque, finanças e operações comerciais, reunindo tudo em um painel fácil de usar. Com ele, é possível acompanhar pedidos, vendas, estoque, caixa e outras informações importantes para o bom funcionamento do negócio.

Perguntas Frequentes

Posso testar antes de assinar?
Para testar, basta preencher o formulário de demonstração acessando o link: https://demo.d7data.com.br. Após o envio, nossa equipe entrará em contato pelo E-mail e WhatsApp, enviando as credenciais para acessar o ambiente de teste.
Como funciona o processo de cobrança assinaturas?
A cobrança é feita utilizando boletos de forma recorrente mensal. É possível mudar a data de vencimento contactando o suporte, além disso o sistema oferece suporte rápido para mudança e cancelamentos de plano.
Posso acessar em qualquer dispositivo?
Sim, o sistema foi desenvolvido de forma responsiva, ou seja, foi planejado para se adaptar em pequenas e grandes telas de dispositivos.
Precisa de instalar em um computador ou servidor para acessar?
Não, nossos sistemas são hospedados em Cloud Computing, ou seja, não há necessidade de instalação, basta realizar o login entrando pelo navegador de sua escolha.
Consigo trazer meus dados de uma planilha ou sistema antigo?
Sim, oferecemos suporte para migração de dados ao novo sistema. Nossos especialistas realizarão uma análise detalhada para avaliar se o seu negócio está apto para a migração. Para mais informações, entre em contato com nosso suporte.
Posso cancelar a qualquer momento?
Sim, o cancelamento é realizado após 7 dias depois do vencimento da fatura. Então se quiser cancelar é só deixar de pagar o plano. Para informação de retorno de acesso, entre em contato com o suporte.
Emite nota fiscal?
Não, entendemos que a emissão de nota é uma parte crucial de um negocio, entretanto no momento estamos desenvolvendo esta funcionalidade.
